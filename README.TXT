#CHAT APP 
- made by hthomaz with the purpose of study in go, websockets, non-relational databases, heroku, and react

- commands:
/count            => returns the numbers of active users in chat
/color "color"    => change color of messages
/flood            => send 3 messages
/whisp_"Username" => send message only visible to you and the destination username
